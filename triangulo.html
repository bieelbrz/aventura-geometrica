Base de um Triângulo - Fórmula

A base de um triângulo é um dos lados do triângulo sobre o qual a altura é medida. É um elemento fundamental na geometria e é utilizado em várias fórmulas relacionadas aos triângulos. Normalmente, a base é representada pela letra "b". Podemos usar a base para calcular a área do triângulo, além de ser utilizada em outras expressões para encontrar medidas relacionadas ao triângulo.

A seguir, vamos conhecer as fórmulas mais comuns do triângulo que se relacionam com a base. Além disso, aprenderemos a calcular o comprimento da base usando essas fórmulas.

Fórmula para a base de um triângulo

A base de um triângulo é um dos lados do triângulo sobre o qual a altura é medida. Usando o comprimento da base, podemos calcular a área do triângulo, bem como outras medidas relacionadas ao triângulo. Da mesma forma, podemos utilizar essas expressões para encontrar a base.

Fórmula da base usando altura

Se tivermos a altura de um triângulo, podemos encontrar o comprimento da base usando a seguinte expressão:

A = b*h/2

onde, P é o perímetro, e a, b, e c são os comprimentos dos lados do triângulo.

Fórmula da base usando área

Podemos usar a área para calcular a base usando a seguinte expressão:

A = √2 * A * h

onde, A é a área do triângulo e h é a altura.
